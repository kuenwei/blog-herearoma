塔羅香氣計劃②晨露芳言 / 2019.2 / 阿卡莎脈輪花精卡

Feb 09, 2019 - 牌卡 每日 晨露芳言 塔羅

在身心靈之中，有很多不同的體系，除了大分類以外，小分類更是多到數不清，阿卡莎花精就是其中一個代表。

有一陣子，我其實不太使用阿卡莎的產品，雖然我陸續也把脈輪油系列都購買齊全，但最後並沒有很大量的使用，記得那是在一個我對身心靈沒有太多的體會了的階段。當然之後接觸更多的身心靈工具，阿卡莎的相關產品就更少在我的生活中出現。

不過「卡牌解讀」一直都是我有興趣且展現天賦的方式，也在最近一年重新看了一下手上的牌卡後，重拾了阿卡莎邁輪花精卡在手上，細細品味。

每張卡片都是個希望種子

「花」通常是植物的生殖器官，當花綻放開始傳遞花粉的時候，種子也將隨著播種而成行，展開新的生命，我都把這一套卡片稱為「阿卡莎花卡」，因為對我來說卡片代表的不是一瓶瓶的花精，而是展現了一朵朵真實的花。

認真看每一張卡片，都會發現有亮光藏在每一張卡片之中，不管今天我們抽出的卡牌是哪一種花，其實都給予了我們一種力量，一種在心中會有一顆種子萌芽、發光。

正因為這樣的特色，花卡總是給我一種「希望感」，因為開始綻放，開始下一代生命孕育的第一站，點點的星光就這樣幻化成一個個可能的未來，而卡片總提醒我們去創造屬於自己的故事。

圖片出自 Pixabay。

透過「美」融化心中的黑

除了講述希望之外，在每一張鮮豔的花卡下，鮮豔的顏色都代表我們正在消融過去的黑暗與不美好，透過花卡的美，心中的黑漸漸的消弭。

如果我們翻閱所附的書籍，其實除了星光種子外，最常看到的就是我們擁抱並接受自己的過去，然後開始放下那些過去對我們的影響，讓傷痛過去，即使我們帶著傷疤，也是能夠癒合之後往前行。

圖片出自 Pixabay。

2019 年 ２ 月的牌卡希望跟你說

今年二月是華人傳統農曆新年，每一年我們總是幫房子打掃，好好的進行「除舊佈新」的計畫。今年希望透過阿卡莎脈輪花精卡的每日陪伴，也讓大家有機會在心情上除舊佈新。

就好像我上面對於花卡的體會，我們總是能在生活中，發現帶來轉變星光種子，同時也獲得支持與勇氣，帶領我們放下那些曾經的傷害與障礙，近一步在黑暗之中看到光明而前行。

這個月，讓我們不僅迎接新年，也嘗試找到自己生活的新處方。
